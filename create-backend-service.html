🚀 Create Backend Service on Render

⚠️ Backend Service Nahi Mili

Render dashboard mein backend service nahi hai. Pehle create karni padegi.

📋 Step 1: Render mein Login

1. Render.com pe jaiye

2. Sign Up ya Log In kariye

3. GitHub account se login karna best hai

🆕 Step 2: New Web Service Create Kariye

1. Dashboard mein New + button click kariye

2. Web Service select kariye

3. Build and deploy from a Git repository choose kariye

🔗 Step 3: GitHub Repository Connect Kariye

1. Connect GitHub click kariye

2. Repository search kariye: ankisho-billing-software

3. Connect button click kariye

Repository: rathi101/ankisho-billing-software Branch: main

⚙️ Step 4: Service Configuration

Service details bhariye:

Name: ankisho-billing-backend Region: Oregon (US West) Branch: main Root Directory: backend Runtime: Node Build Command: npm install Start Command: npm start

Important: Root Directory mein backend likhna zaroori hai!

💰 Step 5: Plan Select Kariye

1. Free plan select kariye

2. Free plan mein ye limitations hain:

  • 750 hours per month
  • Sleep after 15 minutes of inactivity
  • 512 MB RAM

3. Testing ke liye free plan sufficient hai

🔧 Step 6: Environment Variables Add Kariye

Service create karne se pehle ye environment variables add kariye:

NODE_ENV = production JWT_SECRET = ankisho_billing_jwt_secret_2024 JWT_EXPIRES_IN = 30d FRONTEND_URL = https://ankisho-billing-software.netlify.app

Database: Abhi database variable nahi add kariye, pehle service create kariye

✅ Step 7: Create Service

1. Create Web Service button click kariye

2. Deployment start hoga (5-10 minutes lagenge)

3. Logs mein deployment progress dekh sakte hain

Expected URL:

https://ankisho-billing-backend.onrender.com

⚠️ First Deployment Will Fail

Pehli deployment fail hogi kyunki database connection nahi hai. Ye normal hai!

Error message aayega:

❌ MongoDB connection error

Solution: Next step mein database add karenge

🗄️ Step 8: PostgreSQL Database Add Kariye

Service create hone ke baad:

1. Dashboard mein New + click kariye

2. PostgreSQL select kariye

3. Database details:

Name: ankisho-billing-db Database: ankisho_billing User: ankisho

4. Create Database click kariye

🔗 Step 9: Database ko Service se Connect Kariye

1. Backend service mein jaiye

2. Environment tab click kariye

3. Database connection string add kariye:

DATABASE_URL = postgres://ankisho:XXXXX@dpg-xxxxx-a.oregon-postgres.render.com/ankisho_billing DB_TYPE = postgresql

4. Save Changes click kariye

🎉 Step 10: Final Deployment

1. Manual Deploy click kariye

2. Logs mein success message dekho:

✅ PostgreSQL connected successfully 🚀 Server running on port 10000

3. Health check test kariye:

https://ankisho-billing-backend.onrender.com/api/health

💡 Troubleshooting Tips

Issue 1: GitHub Repository Not Found

Solution: Make sure repository is public or GitHub account is connected

Issue 2: Build Fails

Solution: Check Root Directory is set to backend

Issue 3: Service Won't Start

Solution: Check Start Command is npm start

Issue 4: Database Connection Fails

Solution: Make sure DATABASE_URL is correct
