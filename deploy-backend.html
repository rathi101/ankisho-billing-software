🚀 Deploy Ankisho Billing Backend

✅ Step 1: GitHub Repository Ready

Your code is pushed to: https://github.com/rathi101/ankisho-billing-software

🗄️ Step 2: Set Up MongoDB Database (5 minutes)

1. Go to MongoDB Atlas

2. Create free account and cluster (M0 tier)

3. Create database user: ankisho-admin

4. Add IP: 0.0.0.0/0 (allow all)

5. Copy connection string (will look like this):

mongodb+srv://ankisho-admin:PASSWORD@cluster0.xxxxx.mongodb.net/ankisho-billing?retryWrites=true&w=majority

🌐 Step 3: Deploy Backend on Render

1. Go to Render.com

2. Sign up with GitHub account

3. Click "New +" → "Web Service"

4. Connect repository: rathi101/ankisho-billing-software

5. Configure:

  • Name: ankisho-billing-backend
  • Root Directory: backend
  • Build Command: npm install
  • Start Command: npm start

⚙️ Step 4: Add Environment Variables in Render

In Render dashboard, add these environment variables:

MONGODB_URI
mongodb+srv://ankisho-admin:YOUR_PASSWORD@cluster0.xxxxx.mongodb.net/ankisho-billing
JWT_SECRET
ankisho_billing_super_secure_jwt_secret_key_2024_production_min_32_chars
FRONTEND_URL
https://ankisho-billing-software.netlify.app
NODE_ENV
production
JWT_EXPIRES_IN
30d

🎉 Step 5: Deploy & Test

1. Click "Create Web Service" in Render

2. Wait for deployment (5-10 minutes)

3. Test your backend: https://your-backend-url.onrender.com/api/health

4. Update frontend to use your backend URL
